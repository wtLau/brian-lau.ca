## Ever find Twitter UI noisy?

Try the minimalistic look of my @Twitter UI

## Steps
1. Install AdBlocker Ultimate Extension https://addons.mozilla.org/en-US/firefox/addon/adblocker-ultimate/
2. Go to the extension Setting page
3. Go to "Custom Rules"
4. Copy code from www.brian-lau.ca/snippets/minimal-twitter
5. Paste it in "Custom Rules"
